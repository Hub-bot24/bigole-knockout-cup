# BIGOLE Knockout Cup — One Screen

Standalone landscape-only Top 8 finals bracket.

## Display rules
- Week 1 always shows the locked league-table points beside each team.
- Weeks 2–4 show that matchup's finals-round score.
- Qualifying-final winners skip Week 2 and enter Week 3.
- Qualifying-final losers play in Week 2.
- Elimination-final losers are out.
- Tim is excluded before seeding.

## Data
The page reloads `data/live.json` every 60 seconds. The UI is data-ready; a verified SuperCoach importer is still required for genuine automatic live updates.
